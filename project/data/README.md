# Данные проекта
Датасет: **Intel Image Classification** с Kaggle (https://www.kaggle.com/datasets/puneet6060/intel-image-classification). Автор - Puneet Gupta, лицензия CC0 (открытый датасет).

Содержит изображения ландшафтов, собранные из различных источников (включая данные Intel для соревнований по компьютерному зрению).

---

## Состав

Всего 17 000 изображений, 6 классов:

- **buildings** - здания, архитектура (небоскрёбы, дома, мосты)
- **forest** - леса, деревья (хвойные, лиственные, джунгли)
- **glacier** - ледники, снег (ледяные поля, айсберги)
- **mountain** - горы, холмы (скалы, вулканы, пики)
- **sea** - море, океан (побережье, волны, пляжи)
- **street** - улицы, дороги (городские улицы, шоссе)

Разбивка по выборкам:
- Train: 14 034 изображения
- Test: 3 000 изображений
- Prediction: 7 000 изображений без меток (не используются)

---

## Характеристики

- Размер изображений: преимущественно 150×150 px (встречаются и другие, например 150×105)
- Формат: JPEG
- Цветность: RGB, 3 канала
- Датасет сбалансирован: разница между самым большим и маленьким классом около 15%

---

## Предобработка

В проекте используется следующая предобработка:

1. Resize: 150×150 в 224×224 (требование входа ResNet18)
2. Нормализация: mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225] (стандартные значения ImageNet)
3. Аугментации (только для train):
   - RandomHorizontalFlip(p=0.5)
   - RandomRotation(±15°)
   - ColorJitter(brightness=0.2, contrast=0.2, saturation=0.2)
